# Contributing to Taskflow

Thank you for considering contributing to the Taskflow project! We appreciate your help and look forward to any improvements you can make.

## Guidelines for Contributors
- Please ensure your code follows the project's style guide.
- Write clear, descriptive commit messages.
- For new features, please discuss the changes first by opening an issue.

## Development Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/coltonchrane/taskflow.git
   cd taskflow
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm start
   ```

## Pull Request Process
1. Ensure your feature branch is up-to-date with the `main` branch.
2. Open a pull request and describe the changes in detail.
3. Request a review from the maintainers.
4. Make any requested changes, and update your pull request accordingly.
5. Once approved, the maintainers will merge your changes.

Thank you for contributing!
